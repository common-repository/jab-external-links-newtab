== Description ==
Todos los enlaces externos de tu sitio se abriran en una nueva ventana, ya que este plugin agrega <strong>target="_blank"</strong> a todos estos enlaces.

Con este plugin todos los enlaces externos de tu sitio se abrirán en una nueva ventana, incluso los enlaces en los comentarios de las entradas y las imágenes que tengan algún link externo.


-----------

All external links on your site will open in a new window, because this plugin adds <strong> target = "_blank" </ strong> these links.

With this plugin all external links on your site will open in a new window, even the links in the comments of the entries and pictures that have some external link.

== Installation ==
1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

---------

Debes subir el archivo con el nombre links_externos_blank.php a la carpeta `/wp-content/plugins/` de tu blog, y luego activarlo en el panel de administración.

== Frequently Asked Questions ==

= ¿Solo funciona con enlaces externos? =

Sí, los enlaces internos no son modificados.

= Only works with external links?

Yes, the Internal links are intact.
